DBMS

Database Management Sistem

Adalah aplikasi yang di gunakan untuk me-manage data. Tanpa menggunakan DBMS untuk me-manage data seperti data produk, data customer data penjualan dan masih banyak lagi kita harus simpan dalam bentuk file seperti excel.


DBMS biasanya berjalan sebagai aplakasi server yang digunakan untuk me-managae data kita hanya tinggal memberi perintah ke DBMS untuk melakukan proses management datanya, seperti mengubah, menambah, menghapus dan mengambil data.


Contoh DBMS yang populer seperti MySql, PostgreSql, MongoDB, Oracle dan masih banyak lagi.
